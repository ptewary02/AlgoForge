# AlgoForge

🚀 About the Project
AlgoForge is a full-stack coding challenge platform inspired by LeetCode. It allows users to solve algorithmic problems, submit code, track progress, and sharpen their DSA skills — all in one place.
Whether you're preparing for FAANG interviews or just love problem-solving, AlgoForge gives you the environment to grind smarter.

✨ Features

🧩 Problem Library — Browse problems by difficulty (Easy / Medium / Hard) and tags
💻 Online Code Editor — In-browser editor with syntax highlighting and multi-language support
▶️ Code Execution & Judge — Run and submit code with real-time feedback
🔐 User Authentication — Register, log in, and manage your profile
📊 Progress Tracking — View solved problems and submission history
🏷️ Tag Filtering — Filter by Arrays, Trees, DP, Graphs, and more
📱 Responsive UI — Works seamlessly on desktop and mobile
